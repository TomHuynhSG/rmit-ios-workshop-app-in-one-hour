Software Instructions

Start with Swift Playgrounds

Step 1: Open Spotlight Search

Description: This is the quickest way to find and open applications on your Mac.

Instruction: Press Command + Space bar on your keyboard to bring up the Spotlight Search bar.

Step 2: Open Swift Playgrounds

Description: Use Spotlight to locate and launch the Swift Playgrounds application.

Instruction: Type "Swift Playground" into the search bar, then click on the "Swift Playground App" from the results to open it.

Step 3: Create a New SwiftUI App

Description: From the application's main window, you will begin a new project.

Instruction: Click the Create New App... button to start building a new application.

Step 4: Understand the Interface

Description: The Swift Playgrounds workspace is divided into three main panels that help you build your app.

Instruction:

  • Left Panel - Project Explorer: Browse and manage your project files. Think of it as a table of contents for your app.
  • Center Panel - Code Editor: This is where you write your app's code using Swift and SwiftUI to describe what the screen should show.
  • Right Panel - Live Preview (Canvas): Instantly shows what your app looks like based on your code, helping you see changes in real-time.

Troubleshoot with Swift Playgrounds

Step 1: Identify the Error

Description: Sometimes, you may encounter an error that prevents the app preview from loading, such as being "Unable to delete the contents of ExecutionExtension's container."

Instruction: Click the red error icon in the top right of the code editor to see the specific error message.

Step 2: Quit Swift Playgrounds

Description: To begin troubleshooting, you must first completely close the application.

Instruction: Right-click on the Swift Playgrounds icon in your Mac's Dock and select Quit.

Step 3: Open Self Service

Description: You will use a utility called Self Service, provided by RMIT ITS, to run a troubleshooting script.

Instruction: Click the RMIT icon in the top menu bar, then click the Self Service button to launch the application.

Step 4: Clear the Swift Playground Cache

Description: A common fix for this specific error is to clear the application's cached data, which can become corrupted.

Instruction: In Self Service, click Browse and then All. Find the "Delete Swift Playground Cache" option and click the Run button.

Step 5: Verify the Fix

Description: After the cache has been cleared, you can reopen your project to confirm that the issue is resolved.

Instruction: Launch Swift Playgrounds again and open your app. The app should now build successfully and display correctly in the live preview.
